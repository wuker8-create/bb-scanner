# BB Scanner — Bollinger Middle Touch Scanner

PWA-приложение для iPhone. Сканирует монеты Binance на 15m таймфрейме и оповещает при касании средней линии Боллинджера в условиях бычьего тренда.

## Деплой на Vercel (5 минут)

### Вариант 1 — через браузер (проще всего)

1. Зайди на [vercel.com](https://vercel.com) → Sign Up (через GitHub или email)
2. Нажми **Add New → Project**
3. Выбери **"Deploy from your local project"** или просто перетащи папку `bollinger-scanner` в окно браузера
4. Нажми **Deploy** — через 30 секунд получишь ссылку вида `bb-scanner-xxx.vercel.app`

### Вариант 2 — через Vercel CLI

```bash
npm i -g vercel
cd bollinger-scanner
vercel --prod
```

## Установка на iPhone

1. Открой полученную ссылку в **Safari** (обязательно Safari, не Chrome)
2. Нажми кнопку **Поделиться** (квадрат со стрелкой ↑)
3. Выбери **«На экран "Домой"»**
4. Нажми **Добавить**

Приложение появится на рабочем столе как нативное.

## Включение уведомлений

- Нажми кнопку 🔔 в приложении
- Safari спросит разрешение — нажми **Разрешить**
- Теперь уведомления приходят даже когда телефон заблокирован

> **Требование iOS:** Push-уведомления в Safari работают с iOS 16.4+

## Логика детектора

- **Бычий тренд:** цена > EMA(50) и последние 10 свечей растущие
- **Касание средней BB:** |цена − SMA| / SMA × 100 ≤ порог%
- **Алерт** срабатывает при одновременном выполнении обоих условий
- Повторный алерт по той же монете — только после выхода и повторного касания
- Сканирование каждую минуту, 20 монет параллельно
